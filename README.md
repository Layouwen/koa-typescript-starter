# Koa Typescript Starter

## Quick Start

### Install

```bash
pnpm i
```

### Run

```bash
pnpm dev
```

### Build Js

```bash
pnpm build
```

### Build Pkg

```bash
pnpm build:pkg
```

### Build Docker Image

```bash
pnpm build:docker
```

### Test

```bash
pnpm test
# watch
pnpm test:watch
```
