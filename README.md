# Koa Typescript Starter

## Quick Start

### Install

```bash
pnpm i
```

### Run

```bash
pnpm dev
```

### Build

```bash
pnpm build
```

## Features

- [x] **TypeScript**
- [x] **Eslint**
- [x] **Prettier**
- [x] **Husky**
- [x] **Lint**
